I will use the API from LinkedIn, Facebook, YouTube, and Unsplash to connect with my website.

I want several elements in my website to make a rich experience for my users. The website is catered only to musicans and music professionals. The purpose of LinkedIn API is for users to be able to network. The purpose of Facebook is for users to be able to share content they find interesting with other users. The purpose of YouTube is for users to be able to view videos. Lastly the purpose of Unsplash is for users to be inspired by beautiful photographs about musicianship and careers as a musician.
